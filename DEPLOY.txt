================================================================
  PIJU COIN WEBSITE — HOSTINGER DEPLOYMENT GUIDE
================================================================

STEP 1 — ADD YOUR API KEY (do this before uploading)
================================================================

  Open this file in any text editor:

    api/chat.php

  Line 4 looks like this:
    $ANTHROPIC_API_KEY = getenv('ANTHROPIC_API_KEY') ?: 'YOUR_API_KEY_HERE';

  Replace   YOUR_API_KEY_HERE   with your real Anthropic API key:
    $ANTHROPIC_API_KEY = getenv('ANTHROPIC_API_KEY') ?: 'sk-ant-api03-...';

  Get your key at: https://console.anthropic.com

  Save the file. That's the only edit needed.

================================================================
STEP 2 — UPLOAD TO HOSTINGER public_html
================================================================

  Upload these files/folders directly into public_html:

    index.html
    .htaccess
    css/
    js/
    assets/
    api/              <-- contains only chat.php now

  Make sure they land directly in public_html, not in a subfolder.

================================================================
STEP 3 — DONE
================================================================

  Visit your domain. Click the chat bubble. Test it.

================================================================
TROUBLESHOOTING
================================================================

  Bot says "API key not set..."
    → You didn't replace YOUR_API_KEY_HERE in api/chat.php.
      Re-open the file, add the key, re-upload api/chat.php.

  Bot says "cURL is disabled..."
    → In Hostinger hPanel go to: Hosting > Manage > PHP Configuration
      Make sure "curl" extension is enabled. Save and try again.

  Bot says "Connection failed: ..."
    → Paste the full error message in chat and we'll fix it.

  Site shows a directory listing or 403 error
    → Make sure .htaccess was uploaded. Some FTP clients hide
      dot files — in FileZilla go to Server > Force showing
      hidden files, then re-upload .htaccess.

  Images or video broken
    → Make sure you uploaded the full assets/ folder.
