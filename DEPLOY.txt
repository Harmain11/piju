================================================================
  PIJU COIN WEBSITE — HOSTINGER DEPLOYMENT GUIDE
================================================================

WHAT YOU NEED
  - Hostinger hosting plan (any shared plan works)
  - An Anthropic API key for the chatbot
  - FileZilla (free FTP app) OR Hostinger File Manager

================================================================
  STEP 1 — GET YOUR ANTHROPIC API KEY
================================================================

  1. Go to https://console.anthropic.com
  2. Sign in or create a free account
  3. Click "API Keys" in the left menu
  4. Click "Create Key", give it a name like "PIJU Bot"
  5. Copy the key — it looks like: sk-ant-api03-xxxxx...
  6. Keep it safe, you'll need it in Step 2

================================================================
  STEP 2 — ADD YOUR API KEY TO THE PROJECT
================================================================

  Open this file in any text editor (Notepad, VS Code, etc.):

    api/config.php

  Find this line:
    define('ANTHROPIC_API_KEY', getenv('ANTHROPIC_API_KEY') ?: 'your-anthropic-api-key-here');

  Replace   your-anthropic-api-key-here   with your actual key:
    define('ANTHROPIC_API_KEY', getenv('ANTHROPIC_API_KEY') ?: 'sk-ant-api03-YOUR-KEY');

  Save the file.

================================================================
  STEP 3 — UPLOAD FILES TO HOSTINGER
================================================================

  METHOD A — Hostinger File Manager (easiest, no app needed)
  ---------------------------------------------------------
  1. Log in at https://hpanel.hostinger.com
  2. Click "Files" in the top menu, then "File Manager"
  3. Open the "public_html" folder on the left
  4. Click the "Upload" button (top toolbar)
  5. Upload these files and folders:

       index.html
       .htaccess
       css/          (entire folder)
       js/           (entire folder)
       assets/       (entire folder)
       api/          (entire folder — includes your API key)

  6. Make sure all files land directly inside public_html
     (not inside a subfolder like public_html/piju/)

  METHOD B — FileZilla FTP (for larger uploads)
  ---------------------------------------------
  1. In Hostinger hPanel, go to Files > FTP Accounts
  2. Create an FTP account and note the Host, Username, Password
  3. Open FileZilla (filezilla-project.org — free download)
  4. Enter Host, Username, Password, Port 21 — click Quickconnect
  5. On the right side (server), navigate to /public_html
  6. On the left side (your computer), find your project folder
  7. Drag the files listed above into /public_html

================================================================
  STEP 4 — YOU'RE LIVE!
================================================================

  Visit your Hostinger domain in a browser.
  Click the chat bubble in the bottom-right corner.
  Type "What is PIJU?" to test the bot.

  That's it! No server setup, no command line needed.

================================================================
  TROUBLESHOOTING
================================================================

Problem: Chat bubble shows "Network error"
  Fix: Open api/config.php and make sure your API key is there
       (not the placeholder text). Re-upload the file.

Problem: Chat bubble shows "API key not configured"
  Fix: Same as above — the placeholder wasn't replaced.

Problem: Site loads but images/video are broken
  Fix: Make sure you uploaded the full assets/ folder,
       and that files are in public_html (not a subfolder).

Problem: Chatbot worked before but stopped
  Fix: Your Anthropic API key may have run out of credits.
       Go to console.anthropic.com and top up your balance.

Problem: Page shows a directory listing or 403 error
  Fix: Make sure .htaccess was uploaded. Some FTP clients
       hide "dot files" — enable "Show hidden files" in FileZilla
       (Server menu > Force showing hidden files).
