================================================================
  PIJU COIN WEBSITE — HOSTINGER DEPLOYMENT GUIDE
================================================================

STEP 1 — ADD YOUR API KEY (only 1 file to edit)
================================================================

  Open this file in any text editor:

    js/chatbot.js

  The very first line looks like this:
    const ANTHROPIC_API_KEY = "YOUR_API_KEY_HERE";

  Replace   YOUR_API_KEY_HERE   with your real key:
    const ANTHROPIC_API_KEY = "sk-ant-api03-...";

  Get your key at: https://console.anthropic.com
  Save the file.

================================================================
STEP 2 — UPLOAD TO HOSTINGER public_html
================================================================

  Upload these files/folders directly into public_html:

    index.html
    .htaccess
    css/
    js/           <-- chatbot.js contains your API key
    assets/

  That's it. No api/ folder needed anymore.
  No PHP. No server setup. Pure HTML + JavaScript.

================================================================
STEP 3 — DONE
================================================================

  Visit your domain. Click the chat bubble. Test it.

================================================================
TROUBLESHOOTING
================================================================

  Bot says "API key not set..."
    → You didn't replace YOUR_API_KEY_HERE in js/chatbot.js.
      Edit the file, add the key, re-upload js/chatbot.js.

  Bot says "API error: invalid_api_key"
    → The key you entered is wrong or has been revoked.
      Go to console.anthropic.com, create a new key, update
      js/chatbot.js line 3 and re-upload.

  Bot says "Network error..."
    → Check your internet connection. Also make sure you
      uploaded the correct js/chatbot.js (with your key).

  Site shows directory listing or 403 error
    → Make sure .htaccess was uploaded. In FileZilla go to
      Server > Force showing hidden files, then re-upload.

  Images or video broken
    → Make sure you uploaded the full assets/ folder.
